# 매크로 브리핑 대시보드 Spec

## 개요

미국채 10년물 금리와 원달러 환율의 최근 7영업일 추이를 차트로 보여주고, 각 지표와 관련된 주목받은 기사를 한국어로 제공하는 공개 웹 대시보드. 사용자는 매일 오전 7시 자동 갱신된 브리핑을 로그인 없이 접근할 수 있다.

## 범위

### 포함

- 미국채 10년물 금리 주간 추이 차트 (7영업일)
- 원달러 환율 주간 추이 차트 (7영업일)
- 금리 관련 영문 기사 최대 2건 — 3줄 요약 + 번역 전문 + 원문 링크
- 환율 관련 한글 기사 최대 2건 — 3줄 요약 + 원문 그대로 + 원문 링크
- 마지막 갱신 시각 표시
- 매일 오전 7시 자동 갱신

### 제외

- 로그인·회원가입 — 공개 서비스로 충분하며 핵심 가치 검증 전에 불필요
- 알림·구독 기능 — 뉴스 발굴 품질 검증 후 2단계
- 코스피·유가 등 추가 지표 — 지표 2개로 루프 검증 후 확장
- 사용자 맞춤 설정 — 퍼스널라이제이션은 core value 증명 후

## 시나리오

### 1. 정상 접속 — 미국채 10년물 금리 섹션

- **Given** — 오전 7시 자동 갱신이 완료됐고, 지난 7영업일치 금리 데이터와 영문 기사 1-2건이 수집된 상태
- **When** — 사용자가 대시보드 URL에 접속
- **Then** — 미국채 10년물 금리 주간 추이 차트가 표시되고, 기사 카드마다 언론사명·발행 시각·한국어 3줄 요약·번역된 기사 전문·"원문 보기" 링크가 순서대로 표시된다

성공 기준:
- [ ] 차트에 7개 데이터 포인트(영업일 기준)가 렌더된다
- [ ] 기사 카드에 언론사명이 표시된다 (예: "Bloomberg")
- [ ] 기사 카드에 발행 시각이 표시된다
- [ ] 기사 카드 상단에 한국어 3줄 요약이 표시된다
- [ ] 기사 카드에 한국어로 번역된 기사 전문이 표시된다
- [ ] 기사 카드에 "원문 보기" 링크가 표시된다

### 2. 정상 접속 — 원달러 환율 섹션

- **Given** — 오전 7시 자동 갱신이 완료됐고, 지난 7영업일치 환율 데이터와 한글 기사 1-2건이 수집된 상태
- **When** — 사용자가 대시보드 URL에 접속
- **Then** — 원달러 환율 주간 추이 차트가 표시되고, 기사 카드마다 언론사명·발행 시각·한국어 3줄 요약·한글 원문 기사 내용·"원문 보기" 링크가 순서대로 표시된다

성공 기준:
- [ ] 차트에 7개 데이터 포인트(영업일 기준)가 렌더된다
- [ ] 기사 카드에 언론사명이 표시된다 (예: "한국경제")
- [ ] 기사 카드 상단에 한국어 3줄 요약이 표시된다
- [ ] 기사 카드 본문이 별도 번역 없이 한글 원문으로 표시된다
- [ ] 기사 카드에 "원문 보기" 링크가 표시된다

### 3. 원문 링크 클릭

- **Given** — 대시보드가 표시된 상태이고 기사 카드에 "원문 보기" 링크가 있다
- **When** — 사용자가 "원문 보기" 링크를 클릭
- **Then** — 원문 기사가 새 탭에서 열린다

성공 기준:
- [ ] 링크 클릭 시 현재 탭은 그대로이고 새 탭에서 해당 기사 URL이 열린다

### 4. 갱신 시각 확인

- **Given** — 오전 7시 자동 갱신이 완료된 상태
- **When** — 사용자가 대시보드에 접속
- **Then** — 페이지에 마지막 갱신 시각이 "최종 업데이트: 2026-04-30 07:00" 형식으로 표시된다

성공 기준:
- [ ] 갱신 시각이 날짜와 시간(HH:MM) 형식으로 표시된다

### 5. 지표 데이터 수집 실패

- **Given** — 외부 소스 오류로 금리 또는 환율 데이터 수집이 실패한 상태
- **When** — 사용자가 대시보드에 접속
- **Then** — 해당 지표의 차트 자리에 "데이터를 불러올 수 없습니다" 메시지가 표시되고, 나머지 섹션은 정상 표시된다. 마지막으로 성공한 갱신 시각이 함께 표시된다

성공 기준:
- [ ] 금리 데이터 실패 시 금리 차트 자리에 오류 메시지가 표시된다
- [ ] 금리 데이터 실패가 환율 섹션 표시를 막지 않는다
- [ ] 오류 상태에서도 마지막 갱신 시각이 표시된다

### 6. 관련 기사를 찾지 못한 경우

- **Given** — 모든 티어 언론사에서 해당 지표 관련 기사를 수집하지 못한 상태
- **When** — 사용자가 대시보드에 접속
- **Then** — 해당 지표의 기사 카드 영역에 "현재 관련 기사를 찾지 못했습니다" 메시지가 표시된다. 차트는 정상 표시된다

성공 기준:
- [ ] 기사 없음 상태에서 빈 화면 대신 안내 메시지가 표시된다
- [ ] 기사 없음이 차트 표시를 막지 않는다

## 불변 규칙

**독립적 섹션 렌더링** — 금리 섹션과 환율 섹션은 서로 독립적으로 렌더된다. 한 섹션의 데이터 또는 기사 수집 실패가 다른 섹션을 빈 화면으로 만들지 않는다.

**외부 링크는 새 탭** — 모든 "원문 보기" 링크는 현재 탭을 유지하고 새 탭에서 열린다.

**갱신 시각은 항상 표시** — 데이터 수집 성공·실패 여부와 무관하게 마지막 갱신 시도 시각이 항상 페이지에 표시된다.
